# Практические задачи

Задачки лежат в папке `tasks`, запуск сервера:

```
$ cd tasks
$ npm i
$ npm start
```

По умолчанию, сервер запустится по адресу [http://localhost:8080](http://localhost:8080)

Изменения подхватываются автоматически, перезапускать ничего не нужно
